My Excel Project

Project Overview

This project showcases my proficiency in Excel and my ability to analyze and organize data. It includes a variety of advanced Excel and Mathematical techniques such as:

  • Excel Solver Solutions
  • Machine Scheduling without Penalties
  • Advanced formulas (e.g., Primal Simplex Algorithms, Dual Simplex Algorithms)
  • Machine Scheduling with Penalties
  • Mathematical Sensitivity Analysis- Adding contraints
  • Branch and Bound Simplex Algorithms
  • Task Assignment IP and Hungerian Algorithm
  • Knapsac IP
  • Travelling Salesperson & Heuristics(Symmetric and Asymmetric)

Project Details

I have applied Excel Solver to solve real-life optimization problems, including machine scheduling without penalties and with penalties, utilizing advanced formulas like the Primal Simplex and Dual Simplex algorithms. I have also conducted mathematical sensitivity analysis by adding constraints, applied Branch and Bound Simplex algorithms, and used Task Assignment Integer Programming (IP) and the Hungarian Algorithm for optimal task allocation. Additionally, I have worked with Knapsack IP problems and tackled complex routing problems such as the Traveling Salesperson Problem (TSP), employing both symmetric and asymmetric heuristics. My expertise extends to achieving the most optimal solutions in various areas, including goal programming, diet programming, and transportation programming.
